# Point

Stores an x- and y-coordinate pair in two-dimensional space.

## Attributes

| Attribute | Type | 
|---------- | ---- | 
| [ `x` ](#x) | *number* |
| [ `y` ](#y) | *number* |

### x

The X coordinate of the point.

### y

The Y coordinate of the point.
